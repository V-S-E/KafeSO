Программа для проведения заказов для курсовой.
---------------------------------------------
Для того, чтобы подключться к базе данных,
используйте синтаксис для создания sql-строк подклю-
чени. Пример можно найти bd\Local_primer.txt;
			 bd\Server_primer.txt.
----------------------------------------------
Для копирования базы на сервер:
1.Подключитесь к серверу через MS SQl Server;
2.Нажмите на вкладку базы данных;
3.Выберите "Импорт приложения уровня данных";
4.В пункте "Параметры выбора" выберите флаг
"Импорт с локального диска". Выберите файл
из каталога bd\Backup_for_create\Kafe.bacpac;
5. Действуйте инструкциям мастера импорта при-
ложения уровня данных.
6. При успешном выполнении, повторите п. 1-5
для файла bd\Backup_for_create\Worker.bacpac;
-----------------------------------------------
Worker.mdb - база для авторизации пользователей.
Kafe.mdb - база данных для рассчёта стоимости,
проведения и отмены заказов, а так же для рассчёта
итоговых значений.
